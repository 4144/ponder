Detailed Description

Defines objects that hold a set of tags.

TagHolder defines an interface for accessing custom tags stored into instances.

A tag is an identifier which can be of any supported type (it is stored in a ponder::Value). It can also be associated to a value, either constant (static tags) or evaluated each time the tag is requested (dynamic tags).

// Check tag existence
bool scriptable = tagHolder.hasTag("scriptable");
// Retrieve static tag
int category = tagHolder.tag("category");
// Retrieve dynamic tag whose value depends on an object's state
bool hidden = tagHolder.tag("hidden", object);
